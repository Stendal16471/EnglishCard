[![Python](https://img.shields.io/badge/Python-3.9-blue)](https://python.org)

**EnglishCard** — это полнофункциональный Telegram-бот, который помогает пользователям учить английские слова, вести личный словарь и отслеживать прогресс. Проект разработан в рамках курсовой работы.

## 🤖 Где найти бота

**👉 [@connor_teaches_bot](https://t.me/connor_teaches_bot)** — бот работает 24/7, запущен на хостинге Bothost. Попробуй прямо сейчас!

## 🚀 Функциональность

### 📚 Личный словарь
- Добавление новых слов в формате `английское - перевод`
- Удаление слов из словаря
- Просмотр всех добавленных слов

### 🎯 Тестирование с тремя уровнями сложности
| Уровень | Эмодзи | Описание |
|---------|--------|----------|
| Легкий | 🍏 | Только простые слова (цвета, числа, местоимения) |
| Средний | 🍊 | Смесь простых и сложных слов |
| Сложный | 🌶️ | Все слова, включая редкие |

### 📊 Статистика
- Количество слов в словаре
- Количество правильных ответов
- Процент успешных попыток
- Текущий уровень сложности

## 📦 Установка и запуск локально

1. **Клонировать репозиторий**
   ```bash
   git clone https://github.com/Stendal16471/EnglishCard.git
   ```
   ```bash
   cd EnglishCard
   ```
2. **Создать виртуальное окружение** 
   ```bash
   python -m venv venv
   ```
   ##### Linux/Mac
   ```bash
   source venv/bin/activate
   ```
   ##### Windows
   ```bash
   venv\Scripts\activate
   ```
3. **Установить зависимости**
   ```bash
   pip install -r requirements.txt
   ```
4. **Настроить переменные окружения**
   Создать файл .env в корне проекта:
   ```bash
   BOT_TOKEN=ваш_токен_бота
   ```
5. **Запустить бота**
   ```bash
   python main.py
   ```
